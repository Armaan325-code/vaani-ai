# 📖 About

Vaani AI is a Python-based AI Voice Assistant developed using **Flask**, **Google Gemini AI**, and **SQLite**. It provides both voice and text interaction through a modern web interface and can perform various desktop automation tasks.

# 🚀 Installation

```bash
git clone https://github.com/Armaan325-code/vaani-ai.git
cd vaani-ai
pip install -r requirements.txt
```

---

# ▶️ Run

```bash
python app.py
```

Then open:

```
http://127.0.0.1:5000
```

# 🤖 Vaani AI

Vaani AI is a Python-based desktop voice assistant that can understand voice commands, reply using AI, and perform useful system tasks.

---

## ✨ Features

- 🎤 Voice Input
- 🔊 Voice Output
- 🤖 AI Chat (Google Gemini)
- 🔋 Battery Percentage
- 📸 Take Screenshot
- 🕒 Current Time
- 📅 Current Date
- 🌐 Open Google
- ▶️ Open YouTube
- 📧 Open Gmail
- 💻 Open Notepad
- 🧮 Open Calculator
- 🎨 Open Paint
- 📂 Open File Explorer
- 📥 Open Downloads Folder
- 🔍 Search Google
- 🎬 Search YouTube

---

## 🛠️ Technologies Used

- Python
- Google Gemini AI
- SpeechRecognition
- Edge-TTS
- PyAutoGUI
- psutil
- python-dotenv

---

## 🚀 Installation

```bash
git clone https://github.com/Armaan325-code/vaani-ai.git
cd vaani-ai
pip install -r requirements.txt
```

---

## ▶️ Run

```bash
python main.py
```
